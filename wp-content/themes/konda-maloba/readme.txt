=== Konda Maloba ===

A custom WordPress theme for the Konda Maloba company profile / resort website.

== Description ==

Modular, boilerplate theme structure following the WordPress Template
Hierarchy. Logic lives under inc/, reusable markup lives under
template-parts/, and custom page layouts live under page-templates/.

== File Organization ==

* inc/               Theme logic, one responsibility per file.
* template-parts/    Reusable, includable markup fragments.
* page-templates/    Selectable custom Page Templates.
* assets/            CSS, JS, images, fonts, icons.
* languages/         Translation files (.pot/.po/.mo).
